## Task 1: Foundation & Environment Setup
**Objective:** Build a solid foundation in web technologies and set up a professional development environment.

**Skills & Concepts:**
- HTML5: Structure (`<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`), forms, multimedia, tables.
- CSS3: Inline, internal, external styles, Flexbox, Grid, animations, transitions, media queries, colors, backgrounds, shadows, borders, gradients.
- JavaScript: Variables, loops, functions, arrays, DOM manipulation, event handling, form validation.
- PHP: Syntax, variables, arrays, functions, control structures, forms handling (`$_GET`, `$_POST`), include/require.
- MySQL: Create database & tables, CRUD operations, primary & foreign keys, connecting PHP with MySQL.

**Deliverables:**
- Personal Portfolio Website (HTML, CSS, JS)
- Hosted on GitHub Pages
- Demo video

---

## Task 2: Interactive UI & Frontend Development
**Objective:** Create responsive, interactive, and user-friendly interfaces using HTML, CSS, JS, and Bootstrap.

**Skills & Concepts:**
- Bootstrap 5: Grid system, navbar, carousel, cards, modals, responsive utilities.
- Custom Styling: Color palette, smooth scrolling, hover effects, Google Fonts & icons.
- Form Handling: Login & registration validation, password match, show/hide password toggle.
- AJAX: Fetch data from PHP without reloading (e.g., username/email availability check).
- Mobile-first responsive design and reusable components (navbar, footer).

**Deliverables:**
- Login & Registration UI
- Responsive UI with animations
- Demo video

---

## Task 3: Backend Development & CRUD Operations
**Objective:** Implement backend logic, secure user authentication, and database integration with PHP & MySQL.

**Skills & Concepts:**
- User Authentication: Registration with hashed passwords (`password_hash`), login/logout, role-based access (User/Admin), session management.
- Profile Management: Edit profile, profile picture upload, dynamic display.
- CRUD Operations: Create, Read, Update, Delete users/products/orders via PHP forms.
- Security: Prepared statements (`mysqli_prepare`) to prevent SQL injection, server-side validation, encrypted passwords.

**Deliverables:**
- User Management System with CRUD and authentication
- Demo video

## Task 4: Real-World Full Stack Project
**Objective:** Develop a functional full-stack web application with advanced features.

**Skills & Concepts:**

## Task 5: Capstone Project & Deployment
**Objective:** Independently design, build, and deploy a professional-grade web application.

**Skills & Concepts:**
