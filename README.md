# APEX Internship — Task 1

## Project Overview
Personal Portfolio Website built with **HTML, CSS, JavaScript, PHP, and MySQL**.  
Demonstrates core web technologies and a working contact form connected to a database.

---

## Getting Started

### Requirements
- XAMPP (Apache & MySQL)  
- Git  
- Windows 10 or higher  
- Modern web browser  

#### Installation/How to run

1. Clone the repository:
git clone https://github.com/Sudarsini-VB/APEX_Internship.git

2. Place folder in XAMPP htdocs:
C:\xampp\htdocs\apex_intern\

3. Start Apache and MySQL from XAMPP.

4. Open in browser:
http://localhost/apex_intern/task1_foundation/index.html

##### What’s included

- Responsive portfolio layout
- Contact form with MySQL backend
- Basic JavaScript for DOM and validation
- CSS styling with Flexbox, Grid, and transitions
