## Overview
A responsive, interactive AI-themed dashboard built using **HTML, CSS, JavaScript, and Bootstrap 5**.  
It includes login & registration pages, smooth animations, custom styling, and responsive layouts.

---

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5  
- **Fonts & Icons:** Google Fonts, Font Awesome  
- **Version Control:** Git & GitHub  
- **Deployment:** Cloudflare Pages

## Features
- Responsive **Navbar**, **Cards**, **Carousel**, **Modals**, **Buttons**  
- **Form Validation**, **Password Match & Toggle**, **Dummy AJAX Check**  
- **Smooth Scrolling & Hover Effects**  
- Mobile-first design using Bootstrap Grid System
