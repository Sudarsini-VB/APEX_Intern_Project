## Overview
This project is a **responsive personal portfolio website** developed as part of the **APEX Internship — Task 1**.  
It demonstrates core web technologies including **HTML, CSS, JavaScript, PHP, and MySQL**, along with skills in **Git, GitHub**, and **Cloudflare deployment**.

The project represents the **foundation phase** of the internship, focusing on environment setup, frontend structure, and version control practices.

---

## Features
- Responsive single-page layout  
- Smooth navigation (About, Skills, Projects, Contact)  
- JavaScript-based form validation with success popups  
- Publicly hosted via **Cloudflare Pages**  
- Organized folder structure for maintainability  
- Version-controlled with **Git & GitHub**

## How to Run Locally

### Prerequisites
- [XAMPP](https://www.apachefriends.org/) (Apache & MySQL)
- [Git](https://git-scm.com/)
- Any modern web browser

### Steps
1. **Clone the repository**
   ```bash
   git clone https://github.com/Sudarsini-VB/APEX_Intern_Project.git

2. **Move the project to XAMPP htdocs**
   C:\xampp\htdocs\apex_intern\
   
3. **Start Apache and MySQL from the XAMPP control panel.**

4. **Access the project in your browser**
   http://localhost/apex_intern/task1_foundation/index.html
